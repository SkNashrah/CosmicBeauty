# 💄 Cosmic Beauty

A modern cross-platform beauty and cosmetics shopping application built with **.NET MAUI**, following the **MVVM architectural pattern**.

The project demonstrates mobile application development, reusable UI components, clean project organization, and responsive user experience.

---

## ✨ Features

- 🛍 Product Catalog
- 🔍 Product Details
- ❤️ Wishlist
- 🛒 Shopping Bag
- 💳 Checkout
- ⭐ Product Reviews & Ratings
- ➕ Add Product
- ⚙ Application Settings
- 🔔 Snackbar Notifications
- 📱 Cross-platform UI

---

## Tech Stack

| Technology | Purpose |

## Getting Started

### Requirements

- Visual Studio 2022
- .NET 10 SDK
- MAUI Workload

```
dotnet workload install maui
```

Clone the repository

```
git clone https://github.com/USERNAME/CosmicBeauty.git
```

Open

```
CosmicBeauty.sln
```

Run

```
F5
```
